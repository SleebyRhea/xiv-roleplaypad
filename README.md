# XIV Roleplay Pad

Web based reimplementation of the basic features found in [Wordsmith]("https://github.com/MythicPalette/Wordsmith-DalamudPlugin"), with some extras thrown in.

Forked from [browserpad](https://github.com/browserpad/browserpad), check that out too!

## Features

- Post splitting using the standard `(Cur/Max)` formatting specifier
  - Formatting configured to closely resemble posts from the in-game chatbox of FFXIV
  - In-textbox specifier for newline continuation (`>>`)
  - Configurable auto-conversation of `--` to `—`
  - Spellchecking
  - OOC parenthesiing
  - Chat type selection, including a custom chat type (for /tell, or prefixes)
  - Continuation pipe prefix when typing long single posts (i.e. `/em | contiued from last line.`)
- `tail -f` equivalent functionality for logs generated by the Snooper plugin
  - Easily configurable filters for:
    - Different chat types
    - Individual messages
    - Entire characters
  - Autoscrolling
